# Portfólio Pessoal - Marcelo Evangelista

Este projeto consiste no desenvolvimento de um portfólio pessoal utilizando HTML, CSS e JavaScript, com o objetivo de apresentar informações profissionais, formação acadêmica e projetos desenvolvidos.

## 🔗 Acesse o site

👉 https://marcelopevangelista.github.io/portfolio-marcelopevangelista/

## 📌 Sobre o projeto

O site foi desenvolvido com foco em organização, clareza visual e usabilidade. Ele conta com navegação simples e estrutura dividida em seções.

Além disso, foram implementadas funcionalidades em JavaScript, como validação de formulário e alternância entre tema claro e escuro.

## 🧱 Estrutura do site

O portfólio é composto pelas seguintes seções:

- Sobre mim  
- Formação  
- Portfólio  
- Contato  

## 💼 Projetos apresentados

### Formulário Outgoing UNAERP
Projeto front-end desenvolvido para inscrição de alunos interessados em programas de intercâmbio internacional.

🔗 https://marcelopevangelista.github.io/Formulario-Outgoing-UNAERP/

### Formulário Incoming UNAERP
Projeto front-end voltado para inscrição de alunos estrangeiros na instituição.

🔗 https://marcelopevangelista.github.io/Formulario-Incoming-UNAERP/
